# Rubyist Magazine Reboot

[福岡Rubyist会議03](https://regional.rubykaigi.org/fukuoka03/)での
「Rubyist Magazine Reboot」の発表資料です。

Rubyist Magazineが2年以上止まってしまっているので、
その現状の説明と将来どうなってほしいかについての話をします。

### 表示

    rake

### 公開

    rake publish
